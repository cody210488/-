 打球啊 - 籃球揪團平台

  專案介紹

本專案為使用 ASP.NET Core MVC 開發的籃球揪團平台，
提供球場查詢、活動建立與參與功能，並完成 Azure 雲端部署。



  技術架構

* ASP.NET Core MVC
* Entity Framework Core（Code First）
* ASP.NET Core Identity（登入系統）
* Role-based Authorization（管理員權限）
* Azure App Service（網站部署）
* Azure SQL Database（雲端資料庫）


  功能

* 球場查詢與搜尋
* 個人資料建立、修改、照片上傳
* 選擇球場後可建立活動或參與該球場活動
* 活動留言功能、球場評論
* 後台管理系統（Admin 完善中）
* Web API（開發中）
